# 🔐 API Security Risk Analysis – ownCloud

## 📌 Introduction  
APIs (Application Programming Interfaces) play a crucial role in modern web applications by enabling communication between clients and servers. However, insecure APIs can expose sensitive data and allow unauthorized access. This report focuses on analyzing API security risks in the ownCloud web application using Browser Developer Tools. The objective is to identify potential vulnerabilities, understand their impact, and highlight the importance of securing APIs in real-world applications.
